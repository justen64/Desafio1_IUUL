Meus projetos

Cadastro Cultural

O Cadastro Cultural foi uma iniciativa da Prefeitura de Teresópolis com o objetivo de criar um site onde os artistas locais possam divulgar seu trabalho e ganhar mais visibilidade. Esse foi o meu projeto final na residência em software que fiz no Serratec.

Tecnologias utilizadas

React

React é uma biblioteca JavaScript para a criação de interfaces de usuário, focada em componentes reutilizáveis. Ideal para construir interfaces de usuário rápidas e dinâmicas.

Prós

  • Alta performance devido à Virtual DOM
  • Componentes reutilizáveis
  • Grande suporte da comunidade

Contras

  • Curva de aprendizado complexa
  • Configuração inicial pode ser trabalhosa

JavaScript

JavaScript é uma linguagem de programação usada principalmente para o desenvolvimento web, que permite adicionar interatividade e dinamismo às páginas.

Prós

  • Versatilidade e ampla aplicação no desenvolvimento web
  • Alta demanda no mercado de trabalho
  • Grande suporte da comunidade e muitas bibliotecas

Contras

  • Pode ser difícil de depurar devido a erros silenciosos
  • Flexibilidade pode levar a práticas de codificação inconsistentes

Java

Java é uma linguagem de programação robusta e independente de plataforma, amplamente usada para desenvolvimento corporativo e de sistemas.

Prós

  • Plataforma independente (Java Virtual Machine)
  • Suporte a orientação a objetos
  • Altamente seguro e escalável

Contras

  • Consome mais memória do que linguagens mais novas
  • Sintaxe mais verbosa, o que torna o código mais longo

Cadastro Cultural

RocketStore é um e-commerce de Pokémon, desenvolvido como parte de minha residência em software. No front-end, utilizei React JS para criar uma interface intuitiva e responsiva, permitindo aos usuários navegar e realizar compras de maneira eficiente. No back-end, implementei a lógica de negócios em Java, garantindo segurança e estabilidade na comunicação com o banco de dados. Esse projeto me proporcionou uma experiência prática com o desenvolvimento full stack, integrando tecnologias modernas para oferecer uma plataforma envolvente para os fãs de Pokémon.
